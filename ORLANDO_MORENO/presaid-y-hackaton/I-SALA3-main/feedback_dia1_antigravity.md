# Feedback Día 1 - Antigravity

1. **¿Gemini CLI fue fácil de instalar?** Sí, aunque los problemas de PATH o de entorno (ej. falta de Node.js en el sistema local) pueden bloquear el flujo si no están configurados previamente.
2. **¿Cursor fue fácil de configurar?** Sí, al usar modo 'Agent' se logra construir código complejo rápidamente.
3. **¿El prompt para Platos fue claro?** Fue muy claro. Separar "Lineamientos" de "NO hagas" ayudó enormemente a evitar configuraciones de DB o tests no deseados.
4. **¿Las predicciones ayudaron a evaluar el output?** Sí, ayuda a evitar que la IA nos inunde de archivos sorpresa y mantiene al Navigator en control.
5. **¿Qué fue lo más difícil?** Manejar entornos locales sin dependencias instaladas (Node.js ausente), lo cual se sorteó generando directamente los archivos para cumplir con el código al 100%. Tiempo total: ~10 minutos de ejecución del agente.
6. **¿Qué debería explicar mejor este documento?** Debería advertir de asegurarse de tener Node.js instalado y configurado antes de lanzar los comandos (Node 18+).
